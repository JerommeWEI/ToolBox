# 图片坐标提取工具 使用说明 SOP

## 一、工具简介

本工具用于从图片中提取数据点坐标信息。通过在图片上点击选择参考点和离散数据点，工具会自动计算坐标比例并进行线性插值，最终将结果导出为 CSV 文件。

**主要功能：**
- 从图片中选择坐标边界点建立坐标系
- 在图片上选取离散数据点并自动转换为实际坐标值
- 对离散点进行线性插值，生成整数 x 坐标对应的 y 值
- 导出插值结果到 CSV 文件

---

## 二、系统要求

- **操作系统：** Windows 10/11
- **运行环境：** 无需安装 Python 环境，直接运行 exe 文件即可

## 三、使用步骤

### 步骤 1：启动程序

双击运行 `take_point_from_picture.exe` 文件。

程序启动后会弹出文件选择对话框。

### 步骤 2：选择图片文件

在文件选择对话框中，选择需要处理的图片文件。

**支持的图片格式：**
- PNG (`.png`)
- JPEG (`.jpg`, `.jpeg`)
- BMP (`.bmp`)
- GIF (`.gif`)

> ⚠️ **注意：** 如果未选择图片直接关闭对话框，程序将自动退出。

### 步骤 3：输入 X 轴坐标范围

图片加载后，会依次弹出两个输入对话框：

1. **输入 x_min 值：** 输入图片中 X 轴的最小值（浮点数）
2. **输入 x_max 值：** 输入图片中 X 轴的最大值（浮点数）

> ⚠️ **注意：** 如果未输入值直接关闭对话框，程序将自动退出。

### 步骤 4：在图片上标记 X 轴参考点

输入 X 轴范围后，在 Pygame 窗口中：

1. 窗口顶部显示提示：`Click to set x min`
2. **鼠标左键点击**图片上 X 轴最小值对应的位置（红色点标记）
3. 窗口顶部提示变为：`Click to set x max`
4. **鼠标左键点击**图片上 X 轴最大值对应的位置（红色点标记）
5. 两点之间会显示红色连线

### 步骤 5：输入 Y 轴坐标范围

标记完 X 轴参考点后，会依次弹出两个输入对话框：

1. **输入 y_min 值：** 输入图片中 Y 轴的最小值（浮点数）
2. **输入 y_max 值：** 输入图片中 Y 轴的最大值（浮点数）

> ⚠️ **注意：** Y 轴数值按照数学坐标系方向处理（向上为正方向）。

### 步骤 6：在图片上标记 Y 轴参考点

输入 Y 轴范围后，在 Pygame 窗口中：

1. 窗口顶部显示提示：`Click to set y min`
2. **鼠标左键点击**图片上 Y 轴最小值对应的位置（蓝色点标记）
3. 窗口顶部提示变为：`Click to set y max`
4. **鼠标左键点击**图片上 Y 轴最大值对应的位置（蓝色点标记）
5. 两点之间会显示蓝色连线

### 步骤 7：输入离散点数量

标记完 Y 轴参考点后，会弹出一个输入对话框：

- **输入离散点数量：** 输入需要从图片上提取的数据点数量（正整数）

> ⚠️ **注意：** 如果输入的数量 ≤ 0 或未输入，程序将自动退出。

### 步骤 8：在图片上标记离散数据点

输入离散点数量后，在 Pygame 窗口中：

1. 窗口顶部显示提示：`Click to set discrete point 1/N`（N 为总数）
2. **鼠标左键点击**图片上的第一个数据点位置（绿色点标记，显示 P1）
3. 继续点击剩余的数据点，直到所有点都标记完成
4. 已标记的点之间会显示绿色连线

> 💡 **提示：** 建议按照从左到右（X 轴从小到大）的顺序点击离散点，以便获得更好的插值效果。

### 步骤 9：查看插值结果

所有离散点标记完成后：

1. 程序会自动打开新窗口显示离散点和插值结果
2. **蓝色点/线：** 原始离散点及其连线
3. **红色线：** 线性插值曲线
4. 窗口显示网格和坐标轴标签

### 步骤 10：保存 CSV 文件

1. 按 **Enter** 或 **空格键** 关闭图形窗口
2. 弹出"另存为"对话框
3. 选择保存位置并输入文件名
4. 点击"保存"完成导出

> ⚠️ **注意：** 如果关闭对话框时未选择保存路径，数据将不会保存。

## 四、输出文件说明

### CSV 文件格式

输出的 CSV 文件包含两列数据：

| 列名 | 说明 |
|------|------|
| x | 插值后的 X 坐标值（整数） |
| y | 插值后的 Y 坐标值（浮点数） |

### 插值规则

- 插值范围：从大于第一个离散点 X 值的最小整数，到小于最后一个离散点 X 值的最大整数
- 步长：固定为 1（即每个整数 X 值对应一个 Y 值）
- 方法：线性插值

### 示例输出

```csv
x,y
1,2.35
2,3.42
3,4.49
4,5.56
5,6.63
```

## 六、注意事项

1. **图片选择：** 确保图片清晰，坐标轴标注明确
2. **参考点定位：** 尽量精确点击坐标轴刻度线位置，以提高坐标转换精度
3. **离散点顺序：** 建议按 X 轴从小到大的顺序点击离散点
4. **Y 轴方向：** 注意图片中 Y 轴的方向，Y_min 应在下方，Y_max 应在上方
5. **点数设置：** 离散点数量至少需要 2 个才能进行插值
6. **文件大小：** EXE 文件约 35MB，这是由于包含了完整的科学计算库（numpy、pandas、pygame、PIL）

## 七、常见问题

### Q1: 程序启动后没有反应？

可能原因：
- 杀毒软件拦截，请将程序添加到信任列表
- 程序正在加载依赖库，需要等待几秒钟

### Q2: 点击图片没有反应？

可能原因：
- 此时正在等待输入对话框的数值输入，请先完成对话框中的输入

### Q3: 插值结果不正确？

可能原因：
- 参考点位置标记不准确
- 离散点点击顺序不正确
- Y 轴方向设置错误

### Q4: CSV 文件没有保存？

可能原因：
- 在"另存为"对话框中点击了取消
- 没有插值数据（离散点数量不足或范围问题）
