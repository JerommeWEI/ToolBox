根据用户需求，需要在 `TFD-Voltage-MonarchPCBA-setup.py` 中实现以下功能：

### 2. CWL 最接近匹配
```python
def find_closest_cwl(target, cwl_list):
    """找到最接近 target 的 cwl 索引"""
    min_diff = float('inf')
    closest_idx = 0
    for i, cwl in enumerate(cwl_list):
        diff = abs(cwl - target)
        if diff < min_diff:
            min_diff = diff
            closest_idx = i
    return closest_idx
```
